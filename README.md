# Cadastro-de-usuario
cadastro de usuario com express e mongoDB
COMO BAIXAR O PROJETO

# Clonar o repositório
$ git clone https://github.com/MedeirosCode/Cadastro-de-usuario.git

  # Entrar no diretório
  $ cd CadastroUsuarios

  # Instalar as dependências
  $ npm install

  # Iniciar o projeto
  $ node index.js
  
  # Fazer conexão no DB
  $ LINK: www.mongodb.com
